Smart Stock Chatbot
This is a simple web-based chatbot that provides stock information using the Alpha Vantage API.

Features
- Chat-based UI
- Real-time stock price retrieval
- Sector-based company listing
- Sample mutual fund data

Technologies Used
- HTML
- CSS
- Bootstrap
- JavaScript
- Alpha Vantage API

How to Run
1. Get a free API key from Alpha Vantage
2. Replace YOUR_API_KEY_HERE in index.html
3. Open index.html in your browser
